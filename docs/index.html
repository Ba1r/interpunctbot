Spacing Channels

Interpunct Bot can be used to add spaces to channels. Use the command !space channels to replace all the dashes in channel names with spaces. Or, if you use underscores, use !space channels `_`.

For more help, join the support server.

Errors

Channels could not be given spaces. Maybe interpunct does not have permission to Manage Channels?

This error occurs when there is a problem trying to edit channel names. Check the channel permissions and see if there are overrides set that prevent interpunct from managing these channels. Instructions on setting up permissions from Discord.

If you can't find the problem or believe there might be a bug, ask in the support server.
