Spacing Channels

Inter·punct can be used to add spaces to channels. Use the command !space channels to replace all the dashes in channel names with spaces. Or, if you use underscores, use !space channels `_`.

For more help, join the support server.

Errors

Channels could not be given spaces. Maybe inter·punct does not have permission to Manage Channels?

This error occurs when there is a problem trying to edit channel names. Check the channel permissions and see if there are overrides set that prevent interpunct from managing these channels. Instructions on setting up permissions from Discord.

If you can't find the permissions problem or believe there might be a bug, ask in the support server.

There are no channels to put spaces in!

This error occurs when none of your channels have a dash in them. Inter·punct replaces dashes in channel names with spaces when using the space channels command. Add dashes to your channels and run the command again.

For more help or feature requests, ask in the support server.

Sending Messages to Multiple Channels

Inter·punct can be used to send a message to multiple channels at once. To do this, use the command `ip!send: Your message goes here. #tag #some #channels #to #send #it #to`

For more help, join the support server.

Errors

Maybe inter·punct does not have permission to Read and Send Messages there?

This error occurs when there is a problem trying to send messages. Check the channel permissions and see if there are overrides set that prevent inter·punct from reading or sending messages in these channels. Instructions on setting up permissions from Discord.

If you can't find the permissions problem or believe there might be a bug, ask in the support server.

No channels were tagged!

This error occurs when you did not tag any channels in your message. To tag a channel, type # and then select the channel to tag from the list. You can start typing the name of the channel to select it faster.

For more help, ask in the support server.
